# 工业安全产品选型指南

## 选型概述

本指南帮助您根据企业实际需求选择合适的工控安全产品，实现最优的安全投资回报。

## 需求评估

### 评估维度

| 维度 | 评估要点 | 影响产品 |
|------|---------|---------|
| 行业特性 | 合规要求、安全等级 | 整体方案 |
| 系统规模 | 资产数量、网络复杂度 | 产品容量 |
| 安全现状 | 已有防护、历史事件 | 优先级 |
| 预算范围 | 一次性投入、年度维护 | 配置选择 |

### 行业适用性

| 行业 | 推荐方案 | 重点产品 |
|------|---------|---------|
| 电力能源 | 全面防护 | SIEM + 边界防护 |
| 石油化工 | 重点防护 | 边界防护 + 备份 |
| 智能制造 | 基础防护 | 介质管控 + 边界 |
| 医药制造 | 合规导向 | 审计 + 备份 |
| 水务环保 | 基础防护 | 边界防护 |

## 选型流程

### 步骤一：需求收集

```
□ 行业类型和合规要求
□ 系统规模（资产数量、网络结构）
□ 现有安全措施
□ 安全事件历史
□ 预算范围
□ 项目时间计划
```

### 步骤二：方案设计

1. 资产梳理
2. 风险评估
3. 防护策略设计
4. 产品选型
5. 投资估算

### 步骤三：POC 验证

**POC 测试内容：**
- 功能验证
- 性能测试
- 兼容性测试
- 易用性评估

### 步骤四：商务洽谈

- 产品配置确认
- 服务范围确认
- 价格谈判
- 合同签订

## 常见问题

### Q1: 预算有限，应该优先投入什么？

**优先级建议：**
1. **第一优先**：边界防护（隔离 IT/OT）
2. **第二优先**：移动介质管控
3. **第三优先**：数据备份
4. **第四优先**：态势感知

### Q2: 国产替代如何选型？

**国产化建议：**
- 优先选择有自主知识产权的产品
- 关注是否通过国家安全认证
- 评估技术支持本地化能力

### Q3: 如何评估产品性能？

**性能测试要点：**
| 产品 | 测试指标 |
|------|---------|
| 防火墙 | 吞吐量、延迟、并发 |
| SIEM | EPS、查询响应时间 |
| 备份 | 备份/恢复速度 |

### Q4: 如何选择服务商？

**评估标准：**
- 行业经验（同行业案例）
- 技术能力（认证资质）
- 服务响应（SLA 承诺）
- 本地支持（服务网点）
