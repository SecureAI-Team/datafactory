# AOI 设备选型指南

## 选型流程

```
需求分析 → 产品匹配 → 配置选择 → 方案验证 → 商务确认
```

## 需求分析要点

### 1. 产品特征

| 问题 | 影响选型 |
|------|---------|
| 最小元件尺寸？ | 相机分辨率 |
| 最大板尺寸？ | 视野范围 |
| 是否有 BGA/QFN？ | 是否需要 3D |
| 每日产量？ | 设备数量 |

### 2. 质量要求

| 问题 | 影响选型 |
|------|---------|
| 目标误报率？ | AI 配置 |
| 目标漏检率？ | 精度等级 |
| 是否需要追溯？ | 软件配置 |
| 行业标准？ | 定制功能 |

### 3. 现有条件

| 问题 | 影响选型 |
|------|---------|
| 产线节拍？ | 检测速度 |
| 空间限制？ | 设备尺寸 |
| MES 系统？ | 接口需求 |
| 预算范围？ | 产品版本 |

## 配置推荐

### 场景一：消费电子

**特点**：高速、多品种、成本敏感

**推荐**：AOI8000 专业版
- AI 模块减少误报
- 快速换线支持
- MES 集成

### 场景二：汽车电子

**特点**：高可靠性、零缺陷、追溯

**推荐**：AOI8000 旗舰版
- 3D 焊点检测
- 完整追溯方案
- 双机冗余可选

### 场景三：中小企业

**特点**：预算有限、起步阶段

**推荐**：AOI5000 增强版
- 满足基本需求
- 可后续升级 AI
- 投资回收快

### 场景四：PCB 制造

**特点**：大批量、多工序

**推荐**：混合配置
- 内层：AOI5000
- 成品：AOI8000

## 投资回报计算

### 计算公式

```
年节省 = 人力节省 + 不良品减少 + 产能增益
投资回收期 = 设备投资 ÷ 年节省
```
