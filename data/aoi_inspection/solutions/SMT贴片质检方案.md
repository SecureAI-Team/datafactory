# SMT 贴片产线智能质检方案

## 方案概述

面向 SMT（表面贴装技术）产线的端到端智能质检解决方案，覆盖锡膏印刷、贴片、回流焊全流程。

## 行业背景

### SMT 工艺流程

```
PCB 来料 → 锡膏印刷 → SPI检测 → 贴片 → AOI检测 → 回流焊 → AOI检测 → 测试
```

### 质量问题分布

| 工序 | 缺陷占比 | 主要问题 |
|------|---------|---------|
| 锡膏印刷 | 60% | 少锡、偏移、拉尖 |
| 贴片 | 25% | 偏移、漏贴、极性 |
| 回流焊 | 15% | 虚焊、立碑、桥连 |

## 解决方案

### 三道防线架构

```
┌────────────────────────────────────────────────────────────┐

├─────────────────┬─────────────────┬─────────────────────────┤
│  第一道防线      │  第二道防线      │  第三道防线             │
│  锡膏印刷后      │  贴片后          │  回流焊后               │
│  ┌───────────┐  │  ┌───────────┐  │  ┌───────────────────┐ │
│  │ SPI 检测   │  │  │ Pre-AOI  │  │  │ Post-reflow AOI  │ │
```

### 核心设备配置

#### 回流焊后 AOI（必选）

| 产线类型 | 推荐设备 | 配置 |
|---------|---------|------|
| 高速线（CT<15s）| AOI8000 旗舰版 | 3D + AI |
| 标准线（CT 15-30s）| AOI8000 专业版 | AI |
| 经济线（CT>30s）| AOI5000 增强版 | 基础 AI |

#### 贴片后 Pre-AOI（可选）

适用场景：
- 高单价产品
- 复杂 BGA/QFN 器件
- 质量敏感行业（汽车、医疗）

### 智能联动

#### 数据闭环

```
SPI 数据 ─┐
          ├→ 缺陷关联分析 → 工艺调整建议
AOI 数据 ─┘
```

#### 应用场景

1. **偏移关联**：锡膏偏移 + 元件偏移 → 贴片机补偿
2. **虚焊预测**：锡膏量不足 → 预警潜在虚焊
3. **桥连追溯**：锡膏拉尖 → 跟踪后续桥连
