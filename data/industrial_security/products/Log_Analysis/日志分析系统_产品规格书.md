# 工业日志分析系统 产品规格书

## 产品概述

工业日志分析系统（LogInsight-ICS）是面向工业控制系统的日志采集、存储、分析和审计平台。支持多源异构日志统一管理，提供安全事件关联分析和合规审计能力。

## 核心优势

- **广泛兼容**：支持 500+ 种设备日志格式
- **海量存储**：支持 PB 级日志存储与检索
- **智能分析**：AI 驱动的异常行为检测
- **合规审计**：满足等保、网安法审计要求

## 支持的日志源

### IT 设备
- 网络设备：路由器、交换机、防火墙
- 安全设备：IDS/IPS、WAF、堡垒机
- 服务器：Windows、Linux、AIX
- 数据库：Oracle、MySQL、SQL Server
- 应用系统：Web 服务、中间件

### OT 设备
- PLC：西门子、罗克韦尔、施耐德
- DCS：霍尼韦尔、ABB、横河
- SCADA：各主流品牌
- HMI：组态软件操作日志
- 工业交换机：Hirschmann、Moxa

## 功能模块

### 1. 日志采集
- Agent 方式：部署轻量级采集器
- Agentless：通过协议直接采集
- API 对接：与第三方系统集成
- 文件采集：定时读取日志文件

### 2. 日志解析
- 自动识别日志格式
- 内置 500+ 解析模板
- 自定义解析规则
- 字段提取与标准化

### 3. 存储与检索
| 功能 | 说明 |
|------|------|
| 分布式存储 | 支持 PB 级数据 |
| 数据压缩 | 10:1 压缩比 |
| 全文检索 | 秒级响应 |
| 聚合分析 | 多维度统计 |

### 4. 告警与分析
- 基于规则的告警
- 关联分析引擎
- 异常行为检测
- 威胁狩猎支持

### 5. 可视化与报告
- 仪表盘定制
- 趋势分析图表
- 合规报告生成
- 定时报告推送

## 应用场景

- 安全事件调查取证
- 运维故障排查
- 等保合规审计
- 网络安全态势分析
- 用户行为分析
