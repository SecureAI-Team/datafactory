# 某制药企业 GMP 合规与工控安全建设案例

## 客户背景

- **客户名称**：某上市制药集团
- **行业**：医药制造
- **规模**：年产值 80 亿元，5 个生产基地
- **系统范围**：批次控制系统、环境监控系统、LIMS 系统

## 项目挑战

### 业务需求

1. **GMP 合规**：FDA/NMPA 数据完整性要求
2. **等保合规**：等保 2.0 三级要求
3. **数据安全**：配方、工艺属于商业机密
4. **生产连续**：7×24 不间断生产

### 技术挑战

- 批次控制系统要求审计追溯
- 环境监控数据需防篡改
- 老旧系统（Windows Server 2008）仍在运行
- 跨基地统一安全管理

## 解决方案

### 总体架构

```
┌─────────────────────────────────────────────────────────────────┐
│                        集团安全管理中心                          │
│  ┌──────────────────────────────────────────────────────────┐  │
│  │  SIEM-ICS 3000 企业版（私有云部署）                        │  │
│  │  统一监控  │  合规审计  │  报表中心  │  配置管理           │  │
│  └──────────────────────────────────────────────────────────┘  │
│                            ▲                                    │
│          ┌─────────────────┼─────────────────┐                 │
│          │                 │                 │                 │
│  ┌───────┴───────┐ ┌───────┴───────┐ ┌───────┴───────┐        │
│  │   基地 A      │ │   基地 B      │ │   基地 C      │        │
│  │ AI-Fence ×2  │ │ AI-Fence ×2  │ │ AI-Fence ×2  │        │
│  │ USB-Guard ×3 │ │ USB-Guard ×3 │ │ USB-Guard ×3 │        │
│  │ 日志采集     │ │ 日志采集     │ │ 日志采集     │        │
│  └───────────────┘ └───────────────┘ └───────────────┘        │
└─────────────────────────────────────────────────────────────────┘
```

### GMP 数据完整性保障

| ALCOA+ 要求 | 技术措施 | 产品支持 |
|------------|---------|---------|
| Attributable | 用户身份认证、操作关联 | 堡垒机+审计 |
| Legible | 数据可读、完整记录 | LogInsight 日志 |
| Contemporaneous | 实时记录、时间戳 | NTP 同步 |
| Original | 原始数据保护 | AI-Fence 防篡改 |
| Accurate | 数据准确性校验 | 完整性校验 |

### 等保合规对照

| 等保要求 | 合规措施 | 实现产品 |
|---------|---------|---------|
| 安全区域边界 | IT/OT 隔离 | AI-Fence |
| 安全通信网络 | 通信加密 | VPN |
| 安全计算环境 | 终端防护 | 应用白名单 |
| 安全管理中心 | 集中监控 | SIEM-ICS |
| 恶意代码防范 | 介质管控 | USB-Guard |

### 产品部署

| 产品 | 总部 | 每基地 | 合计 |
|------|------|--------|------|
| SIEM-ICS 3000 | 1套 | - | 1套 |
| AI-Fence 5000 | - | 2套 | 10套 |
| USB-Guard Pro | - | 3套 | 15套 |
| LogInsight-ICS | 1套 | - | 1套 |
| 备份系统 | 1套 | 1套 | 6套 |

## 实施效果

### 合规达标

| 合规项 | 状态 | 说明 |
|--------|------|------|
| GMP 数据完整性 | ✅ | 通过 FDA 检查 |
| 等保 2.0 三级 | ✅ | 5 个基地全部通过 |
| NMPA 计算机化系统 | ✅ | 符合附录要求 |

### 安全效果

| 指标 | 建设前 | 建设后 |
|------|--------|--------|
| 安全事件 | 5次/年 | 0次 |
| 审计发现 | 12项 | 2项（已整改） |
| 数据恢复能力 | 无保障 | RPO=0，RTO=4h |

### 审计支持

- **FDA 检查**：2023 年通过 FDA 现场检查，数据完整性无重大缺陷
- **等保测评**：5 个基地一次性通过三级测评
- **内部审计**：实现自动化合规报告，审计效率提升 80%
